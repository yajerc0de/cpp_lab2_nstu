## 📚 Описание

Данный репозиторий содержит решения трёх задач.  
Каждое задание реализовано на нескольких языках:

- **Задание 1 (Факториал: конечные нули)** – C++, Rust, JavaScript (Node.js), Python  
- **Задание 2 (Азбука Морзе: уникальные слова)** – C++, Rust, JavaScript (Node.js), Python  
- **Задание 3 (Числа с чётной длиной)** – C++, Rust, JavaScript (Node.js), Python, Assembler
